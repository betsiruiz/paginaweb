¿Qué es Eclipse?

Eclipse es un entorno de desarrollo integrado (IDE) muy utilizado para programar en Java. Proporciona herramientas para escribir, compilar, ejecutar y depurar código.

Pasos para crear un proyecto en Java

  1. Abrir Eclipse e ir a File > New > Java Project.
  2. Asignar un nombre al proyecto y hacer clic en Finish.
  3. Dar clic derecho en el proyecto > New > Class.
  4. Escribir un nombre para la clase (ej. HolaMundo) y marcar la opción public static void main.
  5. Escribir el siguiente código:
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola mundo desde Java y Eclipse");
    }
}

Ejecutar el programa

Haz clic derecho sobre el archivo y selecciona Run As > Java Application. Verás el resultado en la consola inferior de Eclipse.

1. Descargar e Instalar Eclipse

Ve a la página oficial de Eclipse: Eclipse Downloads

Descargar la versión adecuada: Si eres nuevo en Java, descarga la opción "Eclipse IDE for Java Developers".

Instalar Eclipse: Después de descargar el instalador, sigue las instrucciones para instalar Eclipse en tu sistema operativo.

2. Instalar Java (JDK)

  1. Descargar JDK: Visita el sitio oficial de Oracle JDK o OpenJDK.
  2. Instalar JDK: Sigue las instrucciones de instalación para tu sistema operativo.
  3. Configurar JAVA_HOME: Asegúrate de configurar la variable de entorno JAVA_HOME.

3. Configurar Eclipse con Java

  1. Abrir Eclipse: Al abrir Eclipse, elige un "workspace".
  2. Configurar JDK: Si Eclipse no detecta el JDK:
    1. Ve a Window > Preferences.
    2. Selecciona Java > Installed JREs.
    3. Agrega el JDK si no aparece.

4. Crear tu Primer Proyecto Java

  1. Crear Proyecto: Selecciona File > New > Java Project.
  2. Nombre: Escribe "MiPrimerProyecto".
  3. Crear Clase: Haz clic derecho en src > New > Class.
  4. Nombre de Clase: Escribe "Main" y marca el método main.

5. Escribir y Ejecutar Código Java

Código de ejemplo para imprimir "¡Hola Mundo!":

public class Main {
    public static void main(String[] args) {
        // Imprimir un mensaje en consola
        System.out.println("¡Hola, Mundo!");
    }
}

6. Ejecutar el Proyecto

Haz clic derecho en Main.java y selecciona Run As > Java Application.

7. Depuración y Errores

Eclipse muestra errores en la consola inferior y permite depurar con puntos de interrupción.

Consejos Adicionales

  • Ctrl + Shift + R: Buscar archivos rápidamente.
  • Ctrl + 1: Corrección rápida.
  • Ctrl + D: Eliminar línea actual.

Plugins: Usa Help > Eclipse Marketplace para agregar más funcionalidades.
